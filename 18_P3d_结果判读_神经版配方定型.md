# P3d 结果判读：神经版配方定型（2026-06-12）

## 1. D1：Q 稳定器消融（η=0.125 最坏漂移点，调度关，40k，3 种子）

| 变体 | floor | drift | qRMSE 末段 | 判读 |
|---|---|---|---|---|
| base | 0.269±0.083 | +0.10 | 0.63 | 漂移跨种子复现 |
| q_anchor 单用 | 0.389±0.044 | +0.20 | 0.33 | ❌ Q 滞后变大、方差反升 |
| q_replay 单用 | 0.327±0.223 | +0.11 | 0.12 | ❌ 种子间极不稳（0.08/0.27/0.62） |
| **q_both** | **0.169±0.032** | **−0.11** | **0.16（压平）** | ✅ 唯一同时压住 floor/漂移/种子方差 |
| λ=0.5 交叉验证 | 0.309±0.136 | +0.15 | 1.26 | ❌ 不修 Q 只调估计器无效，反证机制 |

**两个稳定器单用皆败、合用才成立**：anchor 不带 replay → bootstrap 目标滞后整个锚相位，Q 追不上；replay 不带 anchor → 自指目标在回放池上高强度训练，个别种子正反馈发散。合用 = 目标既新鲜（回放重算）又稳定（锚快照）。λ=0.5 的失败反证：问题不在估计器的方差权重，在 Q 本身——**这正是 13 号文档遗留的"adaptive λ 为何不灵"的最终答案**。

## 2. D3：四臂 200k 终局（与 P3b naive/tabular 同图，figP3d_D3_final.png）

| 臂 | 2–20k | 20–60k | 60–100k | 140–200k | min |
|---|---|---|---|---|---|
| none + const η*=0.0625 | 0.256 | 0.100 | 0.455 | 0.662 | **0.046@70k 后爆炸** |
| q_both + const η* | 0.256 | 0.279 | 0.324 | 0.497 | 0.113@22k 缓慢退化 |
| q_both + frozen@η* | 0.126 | 0.264 | 0.202 | 0.267 | 0.073@18k 冻后回升 |
| **q_both + 持续退火** | 0.126 | 0.264 | 0.202 | **0.177↓** | **0.062@198k 仍在降** |

三个关键事实：

1. **C1 的"小 η 一枝独秀"是假象**：none_const 在 70k 触底 0.046 后爆炸到 0.66。固定 η 下 Q 漂移最终毒化一切——C1 的 60k 视界恰好停在爆炸前。教训记入实验方法论：**floor 测量必须配漂移检测 + 足够长视界**（这是本项目第二次测量视界出错，前一次 P1c-C1 口径）。
2. **路径依赖被 Q 稳定器部分消除但未根治**（frozen 臂冻结后仍回升 0.073→0.27），固定/冻结 η 都不行。
3. **持续退火 + q_both 是唯一全程下降的臂**：200k 时 0.177（窗口均值）/ 0.062（min），与 tabular（0.068）同量级，且斜率仍负。

## 3. 最终裁决：P3b 的"调度病理"是误归因，调度无罪

完整因果链（写进论文 discussion 的版本）：

- naive 调度失败（P3b）≠ 逼近底随 η 退火翻倍（Prop-4 推论的原解读）；
- 真因 = **Q 漂移的方差膨胀**：η 退火 → 策略更新变慢 → 采样分布凝固、Q 网在线遗忘加剧 → σ²(t) 增长 → 噪声底 η·σ²(t)/Bτ 不降反升 → 锚快照存入毒化策略（外环放大）；
- 修复 Q（anchor+replay 双稳定器）后，**理论原本设计的退火调度照常工作**：噪声底逐相位减半，长程持续下降；
- Prop 4 修订形式：$\text{floor}(t) \asymp \max\!\big(\tfrac{\eta\,\sigma^2(t)}{B\tau},\ \tfrac{C\delta}{\eta\tau}\big)$，其中 δ（策略蒸馏）实测可忽略（KL~1e-4–1e-7），σ²(t) 含 Q 误差项 $\propto \mathbb{E}(V-Q)^2/\xi$（Prop 3.1/3.2 的 ε_Q 必须时变化）；Q 稳定器的作用 = 保持 σ²(t)≈σ²(0)。

**神经版最终配方**：超相位退火（η×½、K×2，永不冻结）+ Q-anchor（锚相位快照做 bootstrap 目标）+ Q-replay（转移回放、训练时重算目标）+ 策略蒸馏 ds=20 即可。"锚定一切慢变量"成为算法的统一设计原则——策略锚（理论核心）与 Q 锚（target network 的博弈版）同构。

## 4. 证据等级与剩余风险

- 已建立（多种子）：D1 的消融排序与机制（qRMSE 轨迹直接观测）。
- 单种子待固化：D3 持续退火臂的长程下降——**P3e 进行中**（400k、神经 2 种子 vs tabular，判据：floor 跨超相位阶梯下降、≤2× tabular）。
- 论文叙事增益：失败-修复闭环第 4 次；"深度 RL 的 bootstrap 漂移"与博弈收敛理论通过 λ-估计器方差界精确接口——前作（DREAM/ESCHER 的 Q/v 网络）对此完全隐式。
- 风险移交 P4：真特征编码下 q_replay 的内存/陈旧性、anchor 网络化（EMA）后的等价性。

## 5. 下一步

P3e 通过 → P4 启动：特征编码（牌面/行动序列特征替换 one-hot）+ 锚网络（策略/Q 双 EMA 快照）+ Leduc 神经版（对照表格采样版 1.27@1M 的改善幅度）→ 然后才是大博弈与 VR-DeepDCFR+ 对比。P3e 失败（长程仍抬升）→ 回到 16 号文档止损线，降级神经主张。
