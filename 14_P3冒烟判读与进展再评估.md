# P3 冒烟判读 + idea 进展再评估（2026-06-10）

## 1. 冒烟结果：神经版暴露第一个神经特有现象——且被我们自己的定理解释

**现象**（N1，Kuhn 30k quick）：神经版前 ~12k episodes 与表格版重合（0.16–0.25），随后单调抬升到 ~0.39 并停住；表格版继续降到 ~0.11。N2 中 distill=80 也未幸免。

**诊断（Prop 4 推论，时间线吻合）**：神经误差底 = δ/(1−ρ) ∝ **δ/(ητ)**。超相位调度每次把 η 减半：表格 δ=0 故纯受益；神经的每更新蒸馏误差 δ 固定（20 步 Adam），**η 减半 ⇒ 逼近底翻倍**。首次超相位在 8k episodes，抬升起点 ~12–18k，逐级走高——对上了。

**升级为理论（写进论文的新推论）**：
$$\text{神经版稳态底}\ \asymp\ \max\Big(\underbrace{\tfrac{\eta\sigma^2}{B\tau}}_{\text{噪声底，随 }\eta\downarrow\text{ 降}},\ \underbrace{\tfrac{C\delta}{\eta\tau}}_{\text{逼近底，随 }\eta\downarrow\text{ 升}}\Big),\qquad \eta^*\propto\sqrt{\tfrac{B\,C\delta}{\sigma^2}}.$$
**调度必须匹配**：η 减半时蒸馏精度同步减半（δ∝1/预算近似 ⇒ 预算翻倍），保持 δ/η 常数——或在 η\* 处冻结。DeepCFR 系方法中这种"调度-逼近权衡"完全隐式且不可控；我们这里它是显式可证可调的——又一个差异化点。

**已实现修复**（neural_acfr.py）：`grow_distill=True`（η 减半 ⇒ distill_steps 翻倍，封顶 320）与 `eta_min`（冻结）。验证脚本 `run_phase3b.py`（与正在跑的 phase3 输出文件名不冲突）：B1 四方对比（tabular / naive / matched / frozen），B2 关调度后的纯 Prop-4 底排序 + 漂移检测。

**对正在跑的完整版 N1/N2 的预期**：会在更长horizon 上复现同一抬升（更多次超相位 ⇒ 更高的平台）——这不是浪费，它就是论文里 naive 调度的对照曲线，保留数据。

## 2. idea 进展再评估（诚实全景）

**已建立（证据等级：定量）**
1. 核心机制三件套全部成立：内环线性收缩（无-c 速率，B1 实测 2–5% 内）；锚界 O(τ)（B2）；移动锚不退火 → 机器精度、整体线性（B3/C2，13 个数量级）。
2. 主定理定量形式落地：Luque ρ(τ) 形式 + 近常数误差界常数 a≈37（D3 隔离实验），τ\*≈1/a 实践规则。
3. 采样层：噪声剪刀差 83×/39× vs CFR+（P1A2）；model-free 无平均策略收敛（P1B2）；λ 谱两端点理论 + 双博弈排序反转（P1B2/P1c）。
4. 神经层（新）：失败模式本身被 Prop 4 精确预言并给出修复——理论的"预言-失败-修复"闭环第三次出现（前两次：O1 效用尺度、ε_k 锚步频）。

**已裁决（诚实记录）**
- adaptive λ：表格证据不足以进主文 → 附录启发式，神经版再评估。
- A2L 归约路线：机制性排除（写相关工作）。
- D1 方差标度图：测量设计缺陷（均衡流形 + 预算混淆），修正方案在案、非阻塞。

**尚未建立（剩余风险，按大小排序）**
1. **真特征编码下的神经版**（非 one-hot）：桥接版只验证了"光滑参数化不致病"；泛化编码下 δ 的结构未知——P4 的核心风险。
2. **与 SOTA 的正面对比**（VR-DeepDCFR+/MMD/PPO @ Battleship、Liar's Dice、FHP）：还没打过一场。效率主张（以采样局数计）目前只有 vs OS-MCCFR/CFR+ 的证据。
3. Thm 2(a/b) 的证明细节写全（路径已定，工作量问题）。
4. Leduc 采样版绝对速度仍慢于 OS-MCCFR avg（1M eps 时 1.27 vs 0.27）——神经泛化预期改善，需 P4 验证。

**故事线（当前最强版本）**："我们证明并演示了第一个 last-iterate 线性收敛的 model-free 深度 CFR。它删除了 DeepCFR 脉络的平均策略包袱；它的每一层（内环速率、外环速率形式、噪声底、逼近底）都有闭式刻画且被实验以 ≤5% 误差命中；三个失败模式全部被理论预言并修复。"——理论-实验咬合密度是这篇论文区别于"调参出 SOTA"类工作的核心竞争力。
