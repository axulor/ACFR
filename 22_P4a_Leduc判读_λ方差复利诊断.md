## 2. P4c 仪表诊断（100k 短跑，逐假设裁决）

| 假设 | 臂 | 读数 | 裁决 |
|---|---|---|---|
| 特征网跨信息集干扰 | base 仪表 | interf=0.0024（极小），kl=2e-5 | ❌ |
| Q 训练预算不足 | qplus（3× 预算） | qRMSE 1.04→1.89，floor 无改善 | ❌ |
| **λ=1 IS 方差按深度复利** | lam05 | **min 1.418→0.946，qRMSE 同步改善** | ✅ |
| λ 谱内部最优 | lam025/lam075 | 0.25:min 1.086；0.75:跑步中 | λ*≈0.5 |

**机制（写进论文）**：λ-估计器方差逐层递归 Var(h) ≤ (λ²/ξ)[Var(h′)+R²]（Prop 3.2）。Kuhn 深度 2，λ=1 复利可忽略；Leduc 深度 4 + 效用尺度 13，λ=1 的方差复利 ~(1/ξ)⁴ 量级。λ=0.5 把每层因子砍到 1/4，代价是引入 Q 偏差 (1−λ^D)ε_Q（Prop 3.1）——**神经 Q 的泛化使这个偏差可承受，表格 Q 不行**（P1c：表格版 λ=1 最优、λ=0 最差；神经版反转为 λ*≈0.5）。qplus 的失败反证：Q 误差瓶颈不在训练预算，而在移动策略下的 bootstrap 目标噪声——λ 调低后策略更稳 → Q 目标更稳 → qRMSE 反而比 3× 预算更低，是系统级正循环。

**λ-估计器由此从"组件"升级为"深博弈必需品"**：λ 应随深度/Q 质量设置（实践规则：浅博弈 λ→1，深博弈 λ≈0.5 起步）。这正是 DREAM↔ESCHER 谱系存在的意义，也只有神经 Q 才能解锁——表格版没有这个选项。adaptive λ（13/P1d 的修正信号）在神经 Q 下值得重测，列为后续。

**记分牌（Leduc，400k，last-iterate）**：神经 λ=0.5+冻结 **0.702** < 表格 1.394（2×）< OS-MCCFR last 1.779（2.5×）。与 OS-MCCFR avg（0.378，平均策略拐杖）差距从 λ=1 时代的 7× 缩到 1.9×，且我们的曲线尾部仍在下降。

**冻结 vs 退火的博弈依赖性（写进论文 discussion）**：Kuhn 上持续退火赢（P3d），Leduc 上三冻结赢——噪声大的博弈更早撞到参数噪声底，η* 更高。注意本次冻结是三联动语义（η/K/lr 同冻），与 P3d 失败的旧式冻结（K 仍翻倍、无 lr 联动）不同。统一规则：退火到 η*，且所有慢变量同步冻/退。
