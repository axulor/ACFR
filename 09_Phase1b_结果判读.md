# Phase 1b 结果判读（2026-06-10）—— P1 在 Kuhn 上通过

## 1. P1A2：噪声剪刀差 —— **决定性成功，论文核心图**

60k 迭代终值（last-iterate NashConv）：

| 噪声 σ | CFR+ | A-CFR 恒定 η | **A-CFR 调度版** | 剪刀差 |
|---|---|---|---|---|
| 0.05 | 0.054 | 0.0085 | **0.00065** | **83×** |
| 0.2 | 0.101 | 0.037 | **0.0026** | **39×** |

三层结构完全如预测：CFR+ 的 last-iterate 噪声底是**结构性的**（无机制可降）；恒定 η 的 A-CFR 底是**步长伪影**（∝ ησ²）；调度版（η 逐超相位减半、相位加倍）让底**几何下降**且仍在走低。这就是"强单调收缩 + ε_k 调度可证抗噪"的实验化身——figP1A2 是论文 Figure 1 的有力候选。

## 2. P1B2：model-free 采样 + λ 谱 —— **通过，谱行为符合理论**

400k episodes 终值（Kuhn，last-iterate）：

| 算法 | NashConv |
|---|---|
| OS-MCCFR last | 0.32–0.40（噪声垃圾，预期） |
| OS-MCCFR **avg** | 0.0068–0.0083 |
| sacfr λ=0（ESCHER 端点） | **0.012–0.021** |
| sacfr λ=0.5 | 0.011–0.028 |
| sacfr λ=1（DREAM 端点） | 0.021–0.052 |
| sacfr adaptive | 0.021–0.039 |

结论：(i) 小批量修复彻底生效，所有 λ 变体稳定收敛；(ii) **采样版 last-iterate（~0.015）已进入 OS-MCCFR 平均策略（~0.007）的 2–3 倍以内**，而后者的 last 是它的 20–40 倍——"不要平均策略也能在采样下收敛"的核心主张在表格 model-free 设定下成立；(iii) λ 谱排序符合方差理论：λ=1 方差最大最慢，λ=0 在 Kuhn 上最好——因为 Kuhn 的 Q 表太容易学准，λ=0 的偏差项几乎为零，方差主导一切。

**adaptive 没赢的原因与下一步**：Kuhn 区分不出偏差–方差权衡（Q 太准）。Leduc（9k 个历史、非平稳性更强）才是 adaptive 的主场——λ=0 的偏差底应当显形，adaptive 应当胜出。这正是 P1c 的核心实验，也是 λ-估计器作为论文组件的关键证据。
