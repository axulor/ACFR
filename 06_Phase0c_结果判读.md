# Phase 0c 结果判读（2026-06-10）

## 1. C2：线性 last-iterate 率 vs 锚周期 —— **完全验证，且结构信息丰富**

| K | 整体斜率/iter | 终值 |
|---|---|---|
| 25 | −4.4e-4 | 2.4e-9 |
| 50 | **−8.9e-4** | 4.7e-15（机器精度） |
| 100 | **−8.9e-4** | 6.5e-15（机器精度） |
| 200 | −4.9e-4 | 4.2e-10 |

四条曲线全部是跨 8–13 个数量级的直线；K 存在**内部最优**（50–100），两侧各掉一半速率。这与非精确 PPM 理论完全自洽：内环每步收缩 ln(1+ητ)≈0.039 nats，K=25 时内环只解到 e^{−0.98}≈0.38——精度不足拖慢外环；K=200 则浪费内环迭代。K=50/100 时内环达 e^{−2}~e^{−4}，外环全速。**这张图既是主定理（线性 last-iterate）的证据，也直接给出了 K 的选择规则：K ≈ (2~4)/ln(1+ητ)。**

## 2. C1：Leduc —— 稳定性已修复，瓶颈转移到外环速率

好消息：效用归一修复彻底生效——B4 的振荡消失，η 一路到 0.5 全部稳定，norm 组干净地压住 raw 控制组。
坏消息：所有 norm 配置在 NashConv≈0.4 附近进入极慢下降段，10k 步没有进入 0.3。

**定量诊断**（这不是失败，是速率常数问题）：
- Kuhn（C2，K=50）的外环每相位收缩因子 ≈ e^{−0.044} ≈ 0.957，800 个相位 → 12 个数量级。
- Leduc 本轮 K=200、10k 步只有 **50 个相位**，且测得每相位因子 ≈ 0.997——外环收缩比 Kuhn 慢一个数量级以上。
- 两个候选解释：(i) **相位预算不足 + K 没按规则配**（K=200 对 ητ=0.04 过大，浪费迭代）；(ii) **博弈条件数**：外环线性率的常数由多面体误差界（sharpness）决定，Leduc 的均衡近似简并使 sharpness 小。(i) 可立即修，(ii) 决定理论中常数的博弈依赖性——两者都必须搞清楚，这正是 P0 的职责。

**可调的杠杆（带理论依据）**：外环 PPM 步长是 1/τ ⇒ **更小的 τ = 更大的外环步**；内环收缩是 ln(1+ητ) ⇒ η 顶到稳定边缘可在小 τ 下保持内环速度；K 按 C2 规则自动配。D 轮做 (τ,η) 网格 + 相位对齐诊断，分离 (i) 与 (ii)。

## 3. 阶段性总结（诚实版）

- **核心现象已完整成立**（Kuhn）：固定温度、无退火、无平均化，last-iterate 线性收敛到 1e-15。这是论文的 headline，且参数依赖（收缩率、K 规则、τ 底）全部与理论吻合。
- Leduc 是速率常数工程问题，不是机制问题。预期通过 τ↓、η↑、K 规则化解决大部分；残余的博弈依赖常数将如实写进理论（误差界常数）。
- CFR+（avg）在全反馈下仍然快得多（1.7e-5 @10k）——预期内，全反馈速度不是我们的战场；P1 采样版才是。

## 4. D 轮实验（run_phase0d.py）

Leduc 上 (τ,η) ∈ {(0.05,1.0),(0.1,0.5),(0.1,1.0),(0.2,0.5),(0.2,1.0),(0.5,0.5)}，K 按规则 K=⌈7/ln(1+ητ)⌉ 自动配（内环 ~3 个数量级），15k 迭代，评测与相位边界对齐，输出每配置的相位收缩因子表。判据：找到外环因子 ≤0.99/相位 的配置（15k 步即可看到 NashConv < 0.1）；若全部 >0.995，则确认 sharpness 瓶颈，转向外环加速手段（Halpern 锚、相位内平均作锚等，均有文献支撑）。
