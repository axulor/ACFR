# P1d 结果判读（2026-06-10）

## 1. D3（外环速率归因）—— **理论胜利，O-c 疑点解决**

超大 K（内环解到 ~1e-9）下的相位因子：τ=0.1/0.2/0.5 → 0.9812/0.9912/0.9985。
用 Luque (1984) 的 PPM 速率形式 $\rho(\tau)=\frac{a\tau}{\sqrt{1+a^2\tau^2}}$ 反解误差界常数 $a$（即 $a^2=\rho^2/((1-\rho^2)\tau^2)$）：

| τ | ρ 实测 | 反解 a |
|---|---|---|
| 0.5 | 0.9985 | **36.4** |
| 0.2 | 0.9912 | **37.5** |
| 0.1 | 0.9812 | 50.9 |

**结论**：内环残差排除后，τ=0.2/0.5 的 $a$ 稳定在 ≈37——**Luque 函数形式以近常数的博弈常数成立**。此前（rule-K）反解出的漂移（54/38/26）是内环残差污染的伪影。τ=0.1 的偏离（50.9）可归因于：46 个相位、final_nc 仍 0.18，处于 EB 局部生效前的瞬态。

**对理论的直接后果**：Thm 2(b) 从"存在性主张 + 开放的定量刻画"升级为"**Luque 形式的定量线性率，经验常数 a(Leduc)≈37、a(Kuhn) 待测**"。主定理的写法就此定稿：线性率 + ρ(τ) 闭式 + 实验验证函数形式。这是论文理论-实验咬合的第二个亮点（第一个是 Thm 1 的无-c 速率）。顺带给出实践规则：τ 取大反而外环慢（大 aτ 区 −lnρ≈1/(2a²τ²)），τ 应取在 aτ≈1 附近 ⇒ **τ* ≈ 1/a**——把"τ 怎么选"从玄学变成了有公式的事。

## 2. D1（方差标度律）—— 测量设计翻车，原因本身有价值

dist² 底完全不随 B 变化（~0.8–1.15）。原因找到了：**Kuhn 的纳什均衡是一条连续流形**（赌徒参数 α∈[0,1/3] 的单参数族），到"某一个"参考均衡点的距离不收敛到零——随机动力学沿均衡流形漂移，dist² 测的是"漂到了流形哪一段"，不是方差底。NashConv 底确实随 B 降（0.30→0.17→0.13 @ B=4/16/64），但 B=256 反弹（0.28）是第二个设计错误：episodes 固定时 B=256 只有 937 次更新，根本没到稳态。
**修正设计**（列入待办，非阻塞）：(i) 用 NashConv²（由 EB，∝ 到均衡**集合**的平方距离）；(ii) 更新次数对齐（episodes ∝ B）；(iii) 带确定性偏置 c₀ 的拟合或 B=∞ 对照。理论本身（Thm 3）是标准结果，此图属 nice-to-have。

## 3. D2（adaptive λ 重赛）—— 诚实的否定，按预承诺处理

修复信号后 adaptive 从 1.92 改善到 1.51–1.56，方向正确，但仍未超过 λ=1（1.27）@ 1M eps。原因：表格 Q 在 Leduc 上追踪移动目标的滞后偏差太持久，λ=0 分量长期受罚。
**裁决（按 09 文档的预承诺）**：λ-估计器以"谱 + 两端点理论（Prop 3.1/3.2）+ 双博弈排序反转实验"的形态进论文主文——这部分证据完整且漂亮；adaptive 降级为附录的启发式（"有改善、未定论"），神经版阶段重新评估（函数逼近共享统计强度，Q 偏差结构完全不同，可能翻盘）。
