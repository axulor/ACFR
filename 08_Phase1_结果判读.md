# Phase 1 首轮结果判读（2026-06-10）

（运行很快属正常：Kuhn 极小，单轨迹采样在纯 Python 下每条 ~0.1ms，此前的时长预估偏保守。）

## 1. P1A：噪声鲁棒性 —— 半成功，缺的是步长调度

实测（30k 迭代，Kuhn）：
- 无噪声：A-CFR last 4000 步打到 1e-12；CFR+ last 悬浮 ~0.03 不收敛——**最干净的对照组**。
- 噪声 0.05：A-CFR 噪声底 ~0.01，CFR+ ~0.1 —— **10 倍剪刀差** ✓。
- 噪声 0.2：差距缩到 ~2 倍 —— 因为 η 恒定时 A-CFR 的噪声底 ∝ ησ²，不会自行下降。

诊断：这不是机制缺陷而是缺了 Thm 3 要求的**随机步长调度**（η 几何衰减 + 相位几何加长 ⇒ 噪声底逐相位减半）。CFR+ 没有任何对应机制，其 last-iterate 噪声底是结构性的。加调度后预期出现"阶梯下降 vs 水平线"的定量剪刀差。

## 2. P1B：采样版 —— 失败，方差爆炸，两处修复

现象：四个 λ 变体全部悬浮 0.1–0.5；OS-MCCFR avg 正常降到 ~3e-3（基线行为正确，代码无 bug）。
诊断：(i) 单轨迹逐访问更新 + η₀=2.0 → 单步 logit 踢幅可达 ±8，动力学被噪声主导；(ii) 超相位衰减太慢（300k episodes 只衰减 6 次）。
修复（已实现）：**小批量平均**（B=16 条轨迹的优势估计按信息集平均后再做一次 MD 更新，方差 ÷ 访问次数——这本来就是深度版每迭代批量的表格类比，现在表格版与深度版结构对齐了）+ η₀=0.5 + 更强调度（superphase=4，η×0.5，K×2）。
